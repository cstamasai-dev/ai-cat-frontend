# Cat Data App

A React + TypeScript app for managing cat data: list, search, create, edit, and delete.

## Features
- Table view of cats
- Search by name
- Create and edit via modals
- Delete with a row action
- Temperament badges

## Tech Stack
- React 18
- TypeScript
- CRA (react-scripts)

## Getting Started

Install dependencies:

```sh
npm install
```

## Usage

- Use the search bar to filter the list of cats.
- Click on the buttons in the table to create, edit, or delete cat entries.
- The temperament of each cat is displayed as a colored badge for easy identification.
